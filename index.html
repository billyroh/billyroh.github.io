WaffleJS

A night of code, waffles, and karaoke.

All proceeds go to Girls Who Code.

  • Date & Time

    Wednesday, November 4 at 7 PM

  • Location

    SOMA StrEat Food (in the barn)

About

WaffleJS is a meetup about web technology with a side of waffles and karaoke. We meet on the first Wednesday of every month.

All participants are expected to follow the JSConf Code of Conduct.

Schedule

  • 7:20 PM

    Get some tasty waffles and drinks

  • 7:20 PM · Performer

    Performer

    Maria Mora · @_riamaria

    Opening Performance

    Maria will be performing a karaoke rendition of A Whole New World from The Little Mermaid.

  • 7:30 PM

    Introductions

  • 7:45 PM · Speaker

    Speaker

    Mariko Kosaka · @kosamari

    Kernel and Convolution are Friends (Image Processing with Canvas)

    As web engineers, we take it for granted that the &lt;img&gt; tag displays an image because it’s a "digital photo." Let’s look at what "a Pixel" is and how you can manipulate it for fun with the Canvas API and JavaScript.

  • 8:00 PM · Speaker

    Speaker

    Matt Ranney · @mranney

    Spherical geometry and high performance trees in JavaScript

    Our planet is a sphere, but thinking in 3D is hard. I’ll explain some of the basics of working with location data, how Uber finds the nearest vehicles, and how you can use trees to make everything fast.

  • 8:25 PM

    Intermission

  • 8:35 PM · Performer

    Performer

    Evan Minto · @evanminto

    Live performance

    Evan will be performing some of his original music.

  • 8:45 PM · Speaker

    Speaker

    Nick Morgan · @skilldrick

    Creating sample-based music with the Web Audio API

    The Web Audio API is now a fully featured API for working with audio. I’ll show you how you can use this API to pull apart an existing song into short samples and put it back together to create a new song, with very little code.

  • 9:00 PM · Speaker

    Speaker

    Melinda Lu · @skewleft

    Everything old is new again (hopefully): functional programming in JavaScript

    Let’s see how a programming paradigm from the 1950s can help us write JavaScript in a concise, elegant, and super-productive way. We’ll talk about React, Redux, hot reloading, and time-travel debugging.

  • 9:15 PM · Closing

    Closing

    Gabriel Gilder · @_gjg_

    Open Karaoke

    Join us in open karaoke. Gabriel will be opening.
